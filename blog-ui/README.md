# Blog UI (React)

## Start App
1. Copy repository
2. Open Terminal. Right click on blog-ui, select 'Open In -> Terminal'
3. Type 'npm install' and press enter
3. Type 'npm start' and press enter
4. Visit http://localhost:3000
